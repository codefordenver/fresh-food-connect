# Fresh Food Connect

> View this project's Kanban board for tasks here: [![Stories in Ready](https://badge.waffle.io/codefordenver/fresh-food-connect.png?label=ready&title=Ready)](https://waffle.io/codefordenver/fresh-food-connect)
